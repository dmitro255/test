Лабораторная работа: Основы JavaScript
Цель: Изучение базового синтаксиса JS, работа с консолью браузера, типами данных и управляющими конструкциями.
Задания:
1. Выполнение кода
Консоль: Исполнение console.log() и арифметических операций в DevTools.
Интеграция: Создание index.html и подключение скриптов:
Внутренний код через тег <script>.
Внешний файл script.js.
2. Типы данных и логика
Переменные: Объявление name (string), birthYear (number) и isStudent (boolean).
Условия: Реализация проверки баллов через if-else и prompt().
Циклы: Вывод итераций от 1 до 5 через for.
Контрольные вопросы
Отличия var, let, const: Область видимости (блочная vs функциональная) и возможность переопределения.
Неявное преобразование: Автоматическое приведение типов при операциях (например, 5 + '5' = '55').
Сравнение == и ===: Сравнение только по значению против строгого сравнения (значение + тип).
